# Locação Web

Neste documento é apresentado a solução para um sistema de cadastro de veículos de uma 
locadora de automóveis.

## Url acesso

http://localhost:8080/crud/

### Requisitos

Aqui ferramentas e tecnologias utilizadas

```
- Eclipse Neon
- Tomcat 8.5
- Java 7
- AngularJS 1.6.2
- Bootstrap 3.3.7
- UI-Bootstrap 1.3.2
- Jquery 1.12.4
```
